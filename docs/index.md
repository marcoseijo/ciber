# Normas de Compliance más importantes

A continuación se muestran algunas de las normas de compliance más relevantes en España y a nivel internacional:

| Norma               | Año   | Alcance / Descripción breve                                   |
|--------------------|-------|---------------------------------------------------------------|
| UNE-ISO 37301       | 2021  | Sistema de gestión de compliance integral, guía de implementación y certificación. |
| UNE 19601           | 2025  | Gestión de compliance penal, previene delitos corporativos de alto impacto. |
| UNE 19602           | 2019  | Compliance tributario, asegura el cumplimiento fiscal de la empresa. |
| UNE-ISO 37001       | 2017  | Sistema antisoborno certificable, parte de UNE 19601, previene corrupción y cohecho. |
| ISO 19600           | 2014  | Sistema de gestión de compliance (predecesora de UNE-ISO 37301, no certificable). |
| ISO 31000           | 2018  | Gestión de riesgos, marco internacional aplicable a compliance. |
